# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project features a clean and responsive user interface, allowing two players to enjoy the classic game directly in the browser.

The game includes automatic winner detection, draw handling, and reset functionality to provide a smooth and engaging gameplay experience while showcasing fundamental web development concepts and JavaScript DOM manipulation.
